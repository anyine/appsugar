## Golang SDK版本建议:
```
Go SDK >= 1.9
```

## 获取项目源代码方法. **请确保设置了GOPATH环境变量**
```go
go get github.com/hzwy23/appsugar
```

## appsugar简介
appsugar是一款管理系统基础研发平台，主要技术点：
1. 前端 html，css，javascript，使用到的框架：bootstrap，jQuery
2. 后端 Go
3. 数据库 MySQL，Oracle

## 集成功能介绍
1. 采用jwt形式管理用户连接
2. 高效路由服务，灵活中间件接口
3. Metro UI风格洁面
4. 提供权限管理功能
5. 用户操作日志管理
